## 📌 Overview
This project is a GUI-based Expense Tracker built to help users record and manage their spending.

The application allows users to:
- Store expense data  
- View and organize transactions  
- Track total spending  

It is designed as a full application for future use.

---

## ✨ Features
- ➕ Add new expenses  
- 🗑️ Delete expenses  
- 📋 Display all expenses clearly  
- 📊 Calculate total spending  
- 🖥️ GUI interface for easy interaction
